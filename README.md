# 小说编辑器

桌面端小说写作工具，左侧编辑 + 右侧手机预览，每行15字排版。

## 功能

- **章节管理**：分卷、章节增删改、拖拽排序、折叠展开
- **富文本编辑**：加粗、斜体、下划线、字体颜色、背景颜色
- **手机预览**：iPhone 15 风格，15字/行，首行缩进，标点禁则
- **插图**：支持 jpg/png/gif/webp，自动压缩
- **自动保存**：数据存 localStorage，关闭不丢失
- **导出**：TXT（15字/行）、HTML（手机尺寸）
- **跨电脑迁移**：导出 .novel 文件，U盘拷贝导入
- **三种主题**：日间 / 夜间 / 护眼

## 技术栈

- Electron 33
- Quill.js 2.x
- 纯前端 localStorage 存储

## 开发

```bash
npm install
npm start
```

## 打包

```bash
npm run build
```

输出在 `dist/` 目录。
